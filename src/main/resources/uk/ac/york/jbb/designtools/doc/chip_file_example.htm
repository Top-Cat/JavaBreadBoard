Chip File Example

The following is an example of a chip file.

Chip Text=Seq
Description=Simple Sequential Circuit
Manufacturer=Stephen Halstead
Wide Chip=false
Clocked Chip=true;
Input Pins=inA;inB
Output Pins=out1;out2
Number of States=2
Initial State=0
